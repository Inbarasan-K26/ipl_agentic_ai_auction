**IPL Auction System using Agentic AI**
** Project Overview**

This project simulates an IPL cricket auction system using Agentic AI concepts in Python. In the system, IPL teams act as AI agents that automatically decide whether to bid for a player based on player rating and team budget.
The program auctions multiple players and assigns them to teams. Each team has a limited budget and can only buy players if they can afford them. At the end of the auction, the final team squads are displayed.
This project demonstrates basic Artificial Intelligence agent behaviour using Python.

🎯 **Features**

✅ 10 IPL Teams
✅ Multiple Players with Ratings and Roles
✅ AI-Based Bidding System
✅ Team Budget Management
✅ Automatic Player Allocation
✅ Final Team Squad Display

⚙️ **Technologies Used**
Python
Agentic AI Concept
Random Decision Algorithm
